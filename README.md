# 🛒 Quantium Retail Analytics Job Simulation

An end-to-end data analytics and commercial insight project completed as part of the **Quantium Data Analytics Virtual Experience**. This project covers customer segmentation, experimentation, statistical uplift testing, and business recommendations for the Category Manager of Chips.

---

## 📌 Project Overview

This repository is structured into three main tasks:

### 1. Task 1: Customer Analytics & Behavior
* **Objective:** Analyze transaction and purchase data to identify high-value customer segments and buying behaviors.
* **Key Findings:** 
  * **Mainstream Young Singles & Couples** represent the largest sales volume and demonstrate a higher average spend per unit ($3.95).
  * Strong brand affinity for **Kettle** chips and **175g** pack sizes.

### 2. Task 2: Experimentation & Store Trial Assessment
* **Objective:** Evaluate the performance of trial store layout changes implemented in Stores 77, 86, and 88 against baseline Control Stores (233, 155, and 237).
* **Methodology:** Selected control stores using Pearson Correlation and Magnitude Distance metrics over the pre-trial period (Jul 2018 – Jan 2019).
* **Key Findings:** 
  * All three trial stores delivered statistically significant sales uplifts ($p < 0.05$) during the trial period (Feb–Apr 2019).
  * Increased foot traffic and transaction frequency were the primary drivers of growth.

### 3. Task 3: Analytics & Commercial Application
* **Objective:** Translate data findings into actionable, jargon-free commercial insights using the **Pyramid Principle**.
* **Deliverables:** Executive presentation deck and cover email with strategic recommendations for full layout rollout.
